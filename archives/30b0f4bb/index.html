下载你喜欢的B站视频

发表于2026-02-19|更新于2026-02-20|实用教程
|总字数:42|阅读时长:1分钟|浏览量:|评论数:

下载的视频仅供个人观看，请不要盗用B站版权作品。

前言

哔哩哔哩作为中国知名的视频社区，用户规模已经过亿，其中的优质视频非常多。

但是，当我们想下载某个视频到本地时，却找不到下载键。

在之前，我是使用唧唧Down下载B站视频的，但是最近这个工具非常不稳定，而且下载下来的视频画质也被压缩的很低，甚至有时还无法打开视频。我非常苦恼，觉得应该有人和我一样苦恼，总会有大佬自己开发出一个好用的下载工具。于是我就去寻找，终于找到了一个好用的下载工具：DownKyi。
它非常好用，支持所有清晰度，提供工具箱，可下载弹幕文件，封面，音乐，视频等。

Github仓库地址：https://github.com/yaobiao131/downkyicore

接下来直接上教程。

教程

本教程以Windows操作系统为例，其他操作系统教程略有不同。

1.下载工具

这里提供2种下载方式。

Sep’s Cloud by @MC-Sep：A self-hosted file management system with muilt-cloud support.

此方法仅适用于Windows系统！，其他操作系统请前往Github方式。

进入分享链接，找到适合自己系统的zip，下载即可。

Github服务器在国内极不稳定，中国大陆内用户建议不要使用此方法，否则将非常浪费时间。

进入仓库的releases页面，找到最新的版本，找到Assets部分，寻找适于自己系统的文件下载。

由于Github服务器在国内不稳定，可能遇到短时间无法访问的情况。如长时间无法访问，可尝试使用加速器（如Watt Toolkit）。

如果Assets没有展开，要点及Assets左侧的箭头展开文件。

点击名称即可开始下载。

下载速度由于服务器问题可能十分缓慢，需要耐心等待。

下载压缩包后，解压到任意文件夹，找到叫DownKyi.exe的文件（如果找不到，可以在文件资源管理器中Ctrl+F进行搜索），可以创建一个快捷方式到桌面，以便下次进入。

2.下载视频

进入工具后点击右上角头像进行登录，不登录可能无法体验一些功能。点击设置，根据自己的偏好进行设置。
进入哔哩哔哩网页版，找到自己喜欢的视频，点开后复制浏览器链接，粘贴到工具的网址栏，即可对视频进行解析。

粘贴网址后，需要点击视频，点击解析视频，选择画质等基本信息，再点击下载选中项，才可以下载（可以在设置里设置自动解析视频）。

最后

感谢大佬提供那么厉害的工具！
